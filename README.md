# alurabooks
Responsive web project built with HTML5 and CSS3 using the mobile-first approach. Developed during courses at Alura to practice responsive design, layout organization, and frontend development best practices.

## 🚀 Technologies

- HTML5
- CSS3
- Flexbox
- Media Queries

## 📱 Responsiveness

This project was built using the mobile-first methodology, ensuring a better experience across mobile devices, tablets, and desktops.

## 🎯 Project Goal

Practice frontend development concepts such as responsive design, layout organization, and clean code structure.

## 📚 What I Learned

- Semantic HTML structure
- Responsive layouts
- Mobile-first development
- CSS organization
- Flexbox
- Media Queries
